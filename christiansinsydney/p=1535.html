4 February, 2015
讀經材料

一月1日

讀經：創一~二，太一

背經：創一26，太一21, 23

進入事實：

  1. 簡要地說出神在六天的創造中，每一天所造的是什麼（創一2~31）？第七天神又作了什麼（創二1~3）？
  2. 神造人的根據是什麼（創一26~27）？祂如何造人（二7）？造人的目的又是什麼（創一28）？
  3. 神將人安置在伊甸園之後，祂對人有什麼囑咐（創二16~17）？
  4. 神如何建造女人？請敘述這建造的過程，並最後的結果（創二21~25）。
  5. 請敘述耶穌基督的由來。祂是從誰生的？是由什麼懷了孕的（太一16, 18~25）？
  6. 耶穌，這名字的意義是什麼？人稱祂為“以馬內利”，以馬內利是什麼意思（太一21~25）?

一月2日

讀經：創三~五，太二

背經：創三20~21

進入事實：

  1. 當蛇來試誘女人時，牠說了哪些話，至終使人墮落（創三1~7）？
  2. 神對蛇的審判是什麼？對人的管教又是什麼（創三14~19）？
  3. 該隱與亞伯各拿什麼供物給神？神看中的是誰的供物（創四3~9）？
  4. 從誰開始，人開始呼求耶和華的名（創四26）？
  5. 在創世記五章的家譜中，誰與神同行？同行多久？他最後的結局又是什麼（創五22~24）？
  6. 當東方的博士（星象家）來尋找基督時，他們是受什麼指引的？他們又帶什麼禮物獻給那孩子耶穌（太二1~12）？

一月3日

讀經：創六~八，太三

背經：創八21，太三11~12

進入事實：

  1. 神為什麼要毀滅凡有血氣（屬肉體）的人與地？神又吩咐挪亞作什麼，來給人與其他活物預備救恩（創六11~21）？
  2. 方舟建造的尺寸是多少，共分幾層（創六11~21）？
  3. 當洪水退去，挪亞從方舟出來之後，他為耶和華神作了什麼事？耶和華的反應又是如何（創八20~22）？
  4. 施浸約翰傳道的內容是什麼？他又如何介紹那在他以後來的一位（太三1~12）？
  5. 耶穌如何要求施浸約翰為祂施浸？祂受浸之後，聖經如何描述當時的情形（太三13~17）？

一月4日

讀經：創九1~十一9，太四

背經：創九11；太四4, 7, 10

進入事實：

  1. 神與挪亞立約的內容是什麼？立約的記號又是什麼（創九8~17）？
  2. 挪亞醉酒之後，他的兒子含的反應是什麼？閃與雅弗的反應又是什麼（創九22~24）？
  3. 當人在士拿地建造一座城與一座塔時，他們用什麼材料，目的又是為著什麼（創十一1~4）？
  4. 請敘述耶穌受魔鬼的試誘時，他們所說的話，尤其他們所引用的經文（太四1~11）。
  5. 耶穌如何呼召西門和安得烈來跟隨祂（太18~22）？

 

一月5日（周一）

讀經：創十一10~十三，太五1~16

背經：創十二1~3，太五16

進入事實：

  1. 在創世記十一章的家譜中，亞伯蘭的父親與兄弟是誰？他與羅得的關係又是什麼（創十一10~32）？
  2. 耶和華對亞伯蘭的呼召與應許是什麼（創十二1~7）？
  3. 當羅德離開亞伯蘭以後，耶和華對亞伯蘭說了些什麼話，並且要他起來做什麼事（創十三14~18）？
  4. 在創十二~十三這段落中，亞伯蘭在哪些地方，共為耶和華築了幾座壇？
  5. 在耶穌上山對群眾的教訓中，什麼樣的人是有福的？他們所得的福又是什麼（太五1~12）？

一月6日（周二）

讀經：創十四~十六，太五17~48

背經：創十五1，太五44~45

進入事實：

  1. 當亞伯蘭為羅得爭戰得勝回來時，是誰出去迎接他？他祝福的內容是什麼（創十四18~20）？
  2. 所多瑪王願意將財物給亞伯蘭，亞伯蘭的回答是什麼（創十四21~24）？
  3. 亞伯蘭認為是以利以謝承受他的產業，但耶和華對他說，怎樣的人才能成為他的後嗣？亞伯蘭又是如何才被神稱為義（創十五1~6）？
  4. 為什麼亞伯蘭會與夏甲同房生下以實瑪利？夏甲給亞伯蘭生以實瑪利時，亞伯蘭的年歲是多少（創十六）？
  5. 耶穌基督來到地上，祂是如何來看待律法（太五17~20）？
  6. 從殺人、姦淫、與起誓來說，舊約律法的要求與主話的要求有什麼不同（太五21~37）？

一月7日（周三）

讀經：創十七~十八，太六

背經：太六9~13

進入事實：

  1. 在創世記十七章，神與亞伯蘭立約的內容有哪些？立約的記號又是什麼（創十七1~14）？
  2. 神將亞伯蘭與撒萊各改為什麼名字（創十七1~18）？
  3. 亞伯拉罕是在幾歲時受割禮（創十七22~27）？
  4. 當撒拉聽見，明年這時候，她必生一個孩子時，她心裡暗笑。這時候，耶和華如何回覆她（創十八9~15）？
  5. 請敘述亞伯拉罕為所多瑪城向神代求的經過（創十八23~33）。
  6. 主如何教導我們禱告（太六5~15）？

一月8日（周四）

讀經：創十九~二十，太七

背經：太七7~8, 21

進入事實：

  1. 羅得全家離開級將毀滅的所多瑪時，他們果斷嗎？是不是自動並樂意的離開？聖經是如何描述他們的情形？最後天使是因著什麼緣故將他們帶離所多瑪（創十九10~22）？
  2. 羅得的妻子雖然離開了所多瑪城，但是她的結局是什麼？為什麼（創十九23~29）？
  3. 羅得與他兩個女兒亂倫所生的成了哪兩個族類的始祖（創十九30~38）？
  4. 亞伯拉罕為什麼向人說，他的妻子撒拉是他的妹妹？他為亞比米勒禱告，結果是什麼（創二十8~18）？
  5. 律法與申言者的總綱（先知的道理）是什麼（太七7~12）？
  6. 不是每個對主說，主阿，主阿的人就能進諸天的國（天國），那麼是怎樣的人才能進去（太七15~23）？

一月9日（周五）

讀經：創二一~二三，太八1~17

背經：創二二16~18

進入事實：

  1. 亞伯拉罕的兒子以撒出生之後，他為什麼要把夏甲和以實瑪利趕出去（創二一8~13）？
  2. 神試驗亞伯拉罕，要他將以撒獻為燔祭。當他正拿起刀要殺以撒時，為什麼耶和華的使者制止了他（因為他顯出了什麼見證）（創二二9~14）？
  3. 撒拉死了之後，亞伯拉罕將她葬在哪裡？這是亞伯拉罕花了多少錢向誰買的（創二三3~20）？
  4. 當患痲瘋的人前來拜耶穌，並求祂醫治時，主耶穌作了什麼，又說了什麼（太八1~4）？
  5. 為什麼主耶穌稱讚百夫長「這樣大的信心」（太八5~13）？

一月10日（周六）

讀經：創二四1~二五18，太八18~九8

背經：太八20

進入事實：

  1. 當亞伯拉罕的僕人回到米所波大米為以撒娶妻，他是如何向神禱告？他又是如何確定神是否給他亨通的道路（創二四10~27）？
  2. 當亞伯拉罕的僕人確定利百加就是他所尋找的，他對神有什麼回應（創二四42~49）？
  3. 亞伯拉罕一生的年歲是多少？他死後，以撒、以實瑪利將他葬在哪裡（創二五7~11）？
  4. 當一個經學家想來跟隨耶穌時，主的回答是什麼？當門徒中有一位想去埋葬他的父親時，祂的回答又是什麼（太八18~22）？
  5. 當主耶穌與門徒在海上遇見風暴時，祂正在作什麼？當門徒向他求救時，祂的回答又是什麼（太八23~27）？
  6. 主耶穌醫治癱子的目的是要叫人認識什麼（太九1~8）？

一月11日（主日）

讀經：創二五19〜二六35，太九9〜34

背經：創二五23，太九12~13

進入事實：

  1. 利百加所懷的是雙子，他們在母複裡的情形如何？雅各出生時是怎樣的情形，所以被取名叫雅各（創二五22~26）？
  2. 以掃是在怎樣的情形裡將長子的名分賣給雅各？這說出以掃如何看待長子的名分（創二五29~34）？
  3. 當以撒遇見飢荒，往基拉耳去，耶和華向他顯現並說話，神所說的有哪些的重點（創二六1~5）？
  4. 當以撒重新挖掘水井時，他共挖了哪幾口井，每一口井的名字與意思是什麼（創二六15~35）？
  5. 當法利賽人批評主耶穌與稅吏並罪人一同吃飯時，主如何回答（太九9~13）？
  6. 在主耶穌醫治患血漏的女人的事例，那女人在得醫治前心裡所想的是什麼？主對她又說了什麼話（太九18~22）？

一月12日（周一）

讀經：創二七〜二八，太九35〜十15                                                                                  012

背經：創二八12, 17，太九37~38

進入事實：

  1. 雅各用了哪些手段欺騙他父親以撒，使他父親相信他就是長子以掃（創二六5~26）？
  2. 以撒所給雅各的祝福是什麼（創二六27~29）？
  3. 當雅各離家在伯特利過夜時，他夢見什麼？神又藉此向他說了什麼（創二八10~17）？
  4. 當雅各醒來之後，他的感受是什麼？他說了什麼話？作了什麼？並許了什麼願（創二八16~22）？
  5. 當耶穌走遍各城各鄉，祂看見群眾，對他們動了慈心，於是對門徒說了什麼話（太九35~38）？
  6. 當主耶穌打發十二門徒出去傳揚福音時，他們所要作的是什麼（太十5~15）？

 

一月13日（周二）

讀經：創二九1〜三一16，太十16〜十一1                                                                         013

背經：太十38~39

進入事實：

  1. 拉班的兩個女兒中，雅各所愛的是誰？雅各願意為她服事拉班幾年？當雅各服事拉班七年之後，拉班如何欺哄雅各？為了娶到拉結，雅各還得再服事幾年（創二九15~30）？
  2. 請整理雅各前十一個兒子出生的次序，他們的母親，以及他們名字的意義（創二九31~三十24）。
  3. 當拉班願意給雅各牧放群羊的工價時，雅各所要求的是什麼？之後，拉班共改了幾次工價（創三十25~三一12）？
  4. 當主差遣門徒出去，祂說這是如同綿羊在狼的中間，那麼祂又說門徒應當如何來應對呢？若是門徒被交給人的時候，是誰可以在他們裡面說話（太十16~23）？
  5. 當主題到祂在地上所產生的不和時，祂說，人的仇敵就是自己家裡的人。所以，人應當如何的跟從主呢（太十34~39）？

 

一月14日（周三）

讀經：創三一17〜三二21，太十一2〜30                                                                           014

背經：太十一28~30

進入事實：

  1. 當拉般追上雅各之前，神在夢中對他說了什麼？這也叫雅各知道神與他同在，神看見了他的苦情與勞碌（創三一22~42）。
  2. 當雅各行路，神的眾使者遇見他。雅各對他們的認識是什麼？又為此把那地方取名為什麼（創三二1~2）？
  3. 簡單地敘述，當雅各知道以掃帶了四百人迎著他來，他覺得懼怕，所以作了些哪些打算？（為了減少損失，他作了什麼？他向神又如何？為了解以掃的恨，他又是如何安排？）（創三二7~21）
  4. 當約翰差人問耶穌，那將要來的是你嗎？耶穌如何回答（太十一2~6）？
  5. 主責備那些他在其中行過異能，卻沒有悔改的城。但同時，祂又是如何向著父有感謝並頌揚的禱告（太十一20~27）？

 

一月15日（周四）

讀經：創三二22〜三四31，太十二1〜21                                                                           015

背經：太十二20~21

進入事實：

  1. 在雅博渡口，雅各與一個人摔交直到黎明。當那人要離去時，雅各要求什麼，否則就不容他走？那人於是給雅各改什麼新名，原因又是什麼（創三二22~29）？
  2. 雅各將與那人摔交的地方起名叫什麼？為什麼？從那人與雅各的對話中，與之後雅各對那人的認識裡，那與雅各摔交的是誰（創三二27~32）？
  3. 因著底拿被玷汙的事，雅各的兒子們中主要是誰把示劍城中一切的男丁都殺了（創三四25~31）？
  4. 當法利賽人指責主的門徒在安息日掐麥穗來吃，耶穌回答的重點是什麼（太十二3~8）？
  5. 主耶穌醫治了許多人之後，囑咐他們不要顯揚祂，這是應驗了以賽亞所說的什麼話（太十二17~21）？

 

一月16日（周五）

讀經：創三五〜三六，太十二22〜50                                                                                   016

背經：太二二39, 50

進入事實：

  1. 神要雅各上伯特利，在那裡築壇給神，這時候雅各要求他的家人作什麼（創三五1~4）？
  2. 拉結為雅各再生一個兒子，但她難產，臨死的時候，為這兒子取了什麼名字？但雅各卻給他取什麼名字（創三五16~18）？
  3. 以掃遷往西珥山，成了哪一族人的始祖（創三六1~9）？
  4. 當耶穌醫治了一個被鬼附、又瞎又啞的人，法利賽人攻擊祂是憑著什麼趕鬼？但主說，祂若靠著神的靈趕鬼，這就是什麼臨到人的當中（太十二22~28）？
  5. 當幾個經學家和法利賽人要求耶穌行神蹟給他們看，耶穌如何回答（太十二38~40）？

 

一月17日（周六）

讀經：創三七〜三八，太十三1〜30                                                                                     017

背經：太十三14~15

進入事實：

  1. 約瑟所作那兩個夢的內容是什麼（創三七5~11）？
  2. 約瑟的哥哥們要害他，但藉著猶大的建議，約瑟至終被賣給哪一族人，並且被帶到哪裡去（創三七18~28）？
  3. 當猶大知道他瑪是從他懷孕之後，他說了什麼？他瑪所生雙生子的名字又是什麼（創三八24~30）？
  4. 請整理主耶穌所說撒種地比喻中，各種的情形、結果、以及它們的解釋（太十三1~9, 18~23）。

 

一月18日（主日）

讀經：創三九1〜四一39，太十三31〜52                                                                           018

背經：太十三41~43

進入事實：

  1. 當約瑟在波提乏家中作事，為什麼他能凡事亨通，所作的盡都順利？約瑟也因此在主人眼前蒙恩，所以主人派他在家中作什麼（創三九1~6）？
  2. 法老的酒長和司膳長下了監，他們各作了什麼夢？約瑟又如何來解釋（創四十9~19）？

  4. 在諸天的國（天國）好像一粒芥菜種的比喻中，這粒芥菜種長大卻成了什麼？並且有什麼來棲宿在它的枝上（太十三31~32）？
